## ğŸ“– Project Overview

**Komikku** is a high-energy, comic-book-inspired web application built to help users organize their collection of Manga, Manhwa, and Webtoons. The project demonstrates advanced **React CRUD** operations wrapped in a bold **Neo-Brutalist** UI.

### âš¡ Key Features
* **Dynamic CRUD**: Effortlessly add, view, update, and delete comic entries.
* **Instant Previews**: Upload cover art locally and see them rendered instantly using the `FileReader` API.
* **Comic-Pop UI**: A unique aesthetic featuring thick borders, vibrant gradients, and CSS-transformed "burst" elements.
* **Interactive Modals**: Custom-styled confirmation and edit modals that match the comic book theme.
* **Fully Responsive**: Styled with Tailwind CSS to ensure the library looks "pow!" on both mobile and desktop.
